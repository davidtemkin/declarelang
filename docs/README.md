# docs

Browsable documentation, served at `/docs/`.

- `guide/` — hand-authored guide (getting started, concepts, how-to).
- `reference/` — generated API reference, from the `@api` surface of `compiler/`,
  `runtime/`, and `library/`.

The documentation system is specified in [`../design/doc-system.md`](../design/doc-system.md)
(generated reference ⨝ authored guide, with an `@api` ⟺ documented ⟺ public coverage
gate). Not yet built — this is its placed home.
